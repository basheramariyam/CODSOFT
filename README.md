## Task 4: Password Generator
A Python program that generates a strong random password based on the user-defined length.
## Features:
- User inputs desired password length
- Uses letters (uppercase & lowercase), numbers, and symbols
- Generates a random and secure password
## Technologies Used:
- Python
- random module
## How to Run:
1. Run the file `password_generator.py`
2. Enter the required password length
3. Get the generated password as output
